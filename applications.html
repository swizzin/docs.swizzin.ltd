Applications Index

Welcome to the applications documentation repository! In order to categorize things a bit better, applications are now sorted into groups. To your right, you'll see several categories:

  • Automation (Autobrr, Autodl, Bazarr, Lidarr, Medusa, Mylar3, Ombi, Sickchill, Sickgear, Sonarr, Radarr, Prowlarr)
  • Backup & Sync (Resilio, Nextcloud, Rclone, Syncthing, vsftpd)
  • Indexers (Jackett, NZBHydra2)
  • IRC (Lounge, Quassel, ZNC)
  • Media Servers (Airsonic, Calibre-Web, Emby, Jellyfin, Mango, Navidrome, Plex, Tautulli)
  • Torrent Clients & GUIs (Deluge, Flood, qBittorrent, rTorrent, ruTorrent, Transmission)
  • Usenet Clients (NZBGet, SABnzbd, NZBHydra)
  • Utilities (ffmpeg, jfago, Librespeed, Netdata, Pyload, Quota, Rapidleech, Wireguard, X2go, xmrig)
  • Web Features (DuckDNS, Filebrowser, Letsencrypt, Nginx, Organizr, Panel, Shellinabox, Webmin)

Simply navigate to the category and choose the application you're looking for.

If you can't find the application in the list, documentation can generally be accessed directly via /applications/appname, or for example /applications/sonarrv3.

Otherwise, the full list of apps, with a direct link to the docs can be viewed on the homepage
