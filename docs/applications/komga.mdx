Free and open source comics/mangas media server.

## Initial Setup

Installing Komga is easy. Simply issue the following command from SSH:

```bash main
sudo box install komga
```
This command will configure and install Komga for your user.

After installation, the config is stored in `/home/user/.komga/`.

## How to Access

After installation, you can access the web client at `https://<hostname.ltd>/komga/`.
